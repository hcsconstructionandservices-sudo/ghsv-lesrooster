Plaats hier je fotomappen voor de media pagina.

Voorbeelden:
- media_fotos/RtM-dag-2025/
- media_fotos/Bontehondendag-2025/

Aanbevolen bestandsnamen voor foto's: .jpg, .jpeg, .png, .webp

Automatisch miniaturen tonen:
1. Plaats foto's in een map onder media_fotos.
2. Draai ./update-media-index.ps1 (of gebruik push.ps1).
3. Open media -> kies map -> galerij toont miniaturen automatisch.
